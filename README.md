# Amit Sharma - BJP Yuva Politician Profile Website

A modern, responsive personal profile website for Amit Sharma, BJP Yuva Mahamantri of Ward 30, Surat Mahanagar.

## Features

- **Modern Design**: Clean, professional layout with BJP-themed colors (saffron, white, navy blue)
- **Interactive Timeline**: Dedicated page showing political journey from 2018-2024 with photo galleries
- **Complete Gallery Page**: Dedicated gallery displaying all timeline photos with filtering by year
- **Responsive**: Mobile-first design that works on all devices
- **Smooth Scrolling**: Enhanced user experience with smooth navigation
- **Interactive Elements**: Hover effects, animations, and dynamic content
- **Photo Modal System**: Click timeline events or gallery images to view in full-screen carousel
- **Contact Form**: Functional contact form with validation
- **Social Media Integration**: Links to social media profiles

## Sections

1. **Home**: Hero section with profile photo, name, title, and mission statement
2. **About Me**: Political journey and BJP association details
3. **My Journey**: Interactive timeline page with detailed political roadmap
4. **My Work & Camps**: Showcase of various activities including:
   - Blood donation camps
   - Political rallies
   - Election campaigns
   - Social activities
5. **Gallery Preview**: Sample images with link to complete gallery
6. **Complete Gallery**: Dedicated page with all timeline photos (filterable by year)
7. **Contact & Social Links**: Contact information and social media links

## Setup Instructions

1. **Clone or Download** the project files
2. **Add Images**: Replace placeholder image references with actual photos:
   - `assets/bjp-logo.png` - BJP party logo
   - `assets/amit-profile.jpg` - Main profile photo for hero section
   - `assets/amit-about.jpg` - Photo for about section
   - `assets/gallery-1.jpg` to `assets/gallery-6.jpg` - Gallery images
3. **Update Content**: Modify the content in `index.html` to match actual information
4. **Configure Contact**: Update contact information and social media links
5. **Deploy**: Upload to any web hosting service

## Image Requirements

### Required Images:
- **BJP Logo** (`assets/bjp-logo.png`): 200x200px, transparent background
- **Profile Photo** (`assets/amit-profile.jpg`): 500x500px, high quality portrait
- **About Photo** (`assets/amit-about.jpg`): 600x400px, professional photo
- **Gallery Images** (`assets/gallery-1.jpg` to `assets/gallery-6.jpg`): 400x300px each

### Image Guidelines:
- Use high-quality images (minimum 300 DPI for print-ready)
- Optimize for web (compress to reduce file size)
- Maintain aspect ratios as specified
- Use professional, well-lit photographs
